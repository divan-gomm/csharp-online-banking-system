# C# Online Banking System

## Overview
This project was developed as part of my **Higher Certificate in Information Systems (Software Development)** at **Eduvos**.  
It is a **web-based banking system** that allows customers to register, manage accounts, and perform money transfers, demonstrating backend development, database interaction, and service-based architecture concepts.

## Technologies
- **Language:** C#
- **Framework:** ASP.NET Web Forms
- **Web Service:** ASMX
- **Database:** SQL Server
- **Data Access:** ADO.NET, SQL queries, stored procedures

## Features
- Customer registration and login functionality
- Viewing and managing bank accounts
- Secure money transfers between accounts
- Transaction logging and audit trail
- Session-based access control

## Architecture
- **Web Interface:** ASP.NET Web Forms (.aspx, .aspx.cs)
- **Web Service Layer:** ASMX service handling business logic and database operations
- **Data Layer:** Direct SQL queries and stored procedures
- **Database:** SQL Server

## Learning Outcomes
- Developed a multi-page ASP.NET Web Forms application
- Implemented a service layer using ASMX web services
- Worked directly with SQL Server using ADO.NET
- Applied validation, exception handling, and session management
- Gained experience in backend-focused web application development
